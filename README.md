Books
These are not required. However, these books help pave the way from fundamentals to higher levels.

Computer Vision: Algorithms And Applications by Richard Szeliski. An excellent overview of Computer Vision.
Foundations of Computer Vision by Antonio Torralba, Phillip Isola, and William Freeman. Another good overview of Computer Vision
Multiple View Geometry in Computer Vision by Richard Hartley. This book focuses on geometric principles in CV.

Grading and Scoring
Assignments (70%): There are seven (7) biweekly assignments that build on async material and guide you towards practical implementation while solidifying theoretical understanding. These are graded on the following scale

0% Awarded for incomplete or unsubmitted work. This includes any code that does not run; the entire notebook should be executable by 'Running All'.
70% Awarded for a notebook that is mostly correct/complete but demonstrates a lack of conceptual understanding (for example, a key function is incorrect/incomplete).
90% Awarded for good effort and conceptual understanding but the output does not match the provided, objective output (minor typos).
100% Good notebook and matched outputs.
Small point values may also be deducted to call attention to stylistic issues which need to be addressed in future assignments.

Final Project (30%): The final project will draw upon your cumulative understanding of all the course material to produce both well-written code and a report describing your process and results. A rubric for the final project will be provided when it is released.

Late Policy: Students have a total of five (5) late days per semester to use on any assignment(s) with prior approval from the instructor. If you anticipate having trouble completing an assignment, please inform the TA as early as possible ahead of the deadline.

Re-Grading Policy: Students may have until the last week of class to re-submit one homework assignment for re-grading. If you decide to re-submit, please inform the TA once your changes have been submitted.
